Instance description

* Instance type: R (rectangle), L (L-shaped), T (T-shaped)
* Grid square side length
* Plexiglass walls's length
* Orientation of plexiglass walls: HV (horizontal/vertical), DHV (diagonal/horizontal/vertical)
* Matrix of soft obstacles (see Section 3.1). A soft obstacle is represented by a number 0
* List of plexiglass walls. Each line contains two coordinates corresponding to the extremity points of each segment line represent the plexiglass wall.
* List of boundary walls. A boundary wall defines the limits of the room. This information is only displayed for non-regular shaped instances, such as L and T-shaped instances.
